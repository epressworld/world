Hero 区设计方向

提示词要求"图示优先"，Hero 区需要 60% 面积为动态网络图。哪种实现方式？

A

Canvas 动态网络图

8-12个节点实时漂移动画，无中心节点，鼠标悬停高亮。性能好，但SEO不友好。

B

SVG + CSS 动画

静态SVG结构，CSS做节点pulse和连线动画。可缩放，可被搜索引擎索引，但动态性较弱。

C

混合方案

静态SVG作为背景（SEO友好），叠加少量Canvas粒子效果增加生命感。平衡性能和视觉效果。

字体选择

提示词要求 Playfair Display + IBM Plex Mono，但当前项目用的是 Inter + JetBrains Mono

A

按提示词更换字体

Playfair Display（标题）+ IBM Plex Mono（正文）。更editorial风格，有手工感。

B

保留现有字体

Inter + JetBrains Mono。更现代技术感，与现有设计一致。
